# Color Detection App (Streamlit)


A simple Streamlit app to detect colors by clicking on an uploaded image. Shows RGB, closest color name from a dataset, and a color reference box.


## Requirements
- Python 3.8+
- Git (optional, for version control)


## Setup (local)
1. Clone repository (or copy files):
```bash
git clone <your-repo-url>
cd color-detection-app
